HTML Fundamentals

What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a web page using a series of elements that tell the browser how to display the content.

Basic HTML Document Structure

Every HTML document follows a standard structure:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document Title</title>
</head>
<body>
    <!-- Content goes here -->
</body>
</html>

Let's break down the essential parts:

  • <!DOCTYPE html>: Declares the document type and HTML version
  • <html>: The root element of an HTML page
  • <head>: Contains meta information about the document
  • <title>: Specifies a title for the document
  • <body>: Contains the visible page content

HTML Elements & Attributes

HTML elements are represented by tags. Tags are enclosed in angle brackets, and most come in pairs: an opening tag and a closing tag.

<tagname>Content goes here...</tagname>

Attributes provide additional information about HTML elements:

<tagname attribute="value">Content</tagname>

Text Formatting

HTML provides several elements for text formatting:

<h1>This is a Heading 1</h1>
<h2>This is a Heading 2</h2>
<h3>This is a Heading 3</h3>
<p>This is a paragraph.</p>
<strong>Bold text</strong>
<em>Italic text</em>
<mark>Highlighted text</mark>
<br> <!-- Line break -->
<hr> <!-- Horizontal rule -->

Links & Navigation

Links are created using the <a> (anchor) element:

<a href="https://example.com">Visit Example.com</a>
<a href="about.html">About Us</a>
<a href="#section-id">Jump to Section</a>

Images & Media

Images are added using the <img> element:

<img src="image.jpg" alt="Description of image" width="300" height="200">

For video and audio:

<video width="320" height="240" controls>
    <source src="movie.mp4" type="video/mp4">
    Your browser does not support the video tag.
</video>

<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
    Your browser does not support the audio element.
</audio>

Lists & Tables

HTML supports ordered (numbered) and unordered (bulleted) lists:

<ul>
    <li>Unordered item 1</li>
    <li>Unordered item 2</li>
</ul>

<ol>
    <li>Ordered item 1</li>
    <li>Ordered item 2</li>
</ol>
  • Unordered item 1
  • Unordered item 2
  1. Ordered item 1
  2. Ordered item 2

Tables are created using <table>, <tr> (table row), <th> (table header), and <td> (table data) elements:

<table>
    <tr>
        <th>Name</th>
        <th>Email</th>
    </tr>
    <tr>
        <td>John Doe</td>
        <td>john@example.com</td>
    </tr>
    <tr>
        <td>Jane Smith</td>
        <td>jane@example.com</td>
    </tr>
</table>

Forms & Input

Forms are used to collect user input:

<form action="/submit-form" method="post">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name" required>
    
    <label for="email">Email:</label>
    <input type="email" id="email" name="email" required>
    
    <label for="message">Message:</label>
    <textarea id="message" name="message" rows="4" required></textarea>
    
    <input type="submit" value="Submit">
</form>

CSS Basics

What is CSS?

CSS (Cascading Style Sheets) is a styling language used to describe the presentation of a document written in HTML. CSS describes how elements should be rendered on screen, on paper, or in other media.

CSS Syntax

CSS consists of a selector and a declaration block:

selector {
    property: value;
    property: value;
}

Ways to Include CSS

There are three ways to include CSS in your HTML document:

1. External CSS (most common method):

<!-- In the head section of HTML -->
<link rel="stylesheet" href="styles.css">

2. Internal CSS:

<style>
    body {
        background-color: #f0f0f0;
        font-family: Arial, sans-serif;
    }
</style>

3. Inline CSS (use sparingly):

<p style="color: blue; font-size: 18px;">This is a styled paragraph.</p>

CSS Selectors

Selectors define which elements to style. Here are the most common selectors:

Selector Example Description
Element p { } Selects all <p> elements
Class .info { } Selects all elements with class="info"
ID #header { } Selects the element with id="header"
Descendant div p { } Selects all <p> elements inside <div> elements
Child div > p { } Selects all <p> elements that are direct children of <div>
Pseudo-class a:hover { } Selects all <a> elements when mouse hovers over them

The Box Model

The CSS box model is essentially a box that wraps around every HTML element. It consists of: content, padding, border, and margin.

.box {
    width: 300px;        /* Content width */
    padding: 20px;       /* Space between content and border */
    border: 2px solid;   /* Border around the padding */
    margin: 30px;        /* Space outside the border */
}

By default, the width and height properties define the size of the content area. To change this behavior:

* {
    box-sizing: border-box;  /* Width and height include padding and border */
}

Colors & Typography

CSS provides various ways to specify colors:

/* Color Names */
color: red;

/* Hex Codes */
color: #ff0000;  /* Red */
color: #00ff00;  /* Green */
color: #0000ff;  /* Blue */

/* RGB */
color: rgb(255, 0, 0);  /* Red */

/* RGBA (with transparency) */
color: rgba(255, 0, 0, 0.5);  /* 50% transparent red */

/* HSL (Hue, Saturation, Lightness) */
color: hsl(0, 100%, 50%);  /* Red */

/* HSLA (with transparency) */
color: hsla(0, 100%, 50%, 0.5);  /* 50% transparent red */

Typography properties:

body {
    font-family: 'Arial', sans-serif;
    font-size: 16px;
    font-weight: 400;          /* Normal: 400, Bold: 700 */
    line-height: 1.5;          /* 1.5 times the font size */
    letter-spacing: 1px;
    text-align: center;        /* left, right, center, justify */
    text-decoration: underline; /* underline, overline, line-through */
    text-transform: uppercase;  /* uppercase, lowercase, capitalize */
}

Layout & Positioning

CSS offers several ways to position elements:

/* Default positioning */
position: static;

/* Relative to its normal position */
position: relative;
top: 20px;
left: 30px;

/* Relative to the viewport */
position: fixed;
top: 0;
right: 0;

/* Relative to the nearest positioned ancestor */
position: absolute;
bottom: 20px;
right: 30px;

/* Similar to fixed, but relative to nearest scrollable ancestor */
position: sticky;
top: 0;

Responsive Design

Media queries are used to apply different styles for different devices/screen sizes:

/* Base styles for all devices */
body {
    font-size: 16px;
}

/* Styles for tablets and larger */
@media (min-width: 768px) {
    body {
        font-size: 18px;
    }
}

/* Styles for desktops and larger */
@media (min-width: 1024px) {
    body {
        font-size: 20px;
    }
}

Advanced Techniques

Flexbox Layout

Flexbox is a one-dimensional layout method for arranging items in rows or columns:

.container {
    display: flex;
    flex-direction: row;        /* row, column, row-reverse, column-reverse */
    justify-content: space-between; /* flex-start, flex-end, center, space-around, space-between */
    align-items: center;        /* flex-start, flex-end, center, stretch, baseline */
    flex-wrap: wrap;            /* nowrap, wrap, wrap-reverse */
    gap: 20px;                  /* gap between flex items */
}

.item {
    flex: 1;                    /* shorthand for flex-grow, flex-shrink, flex-basis */
}

CSS Grid

CSS Grid is a two-dimensional layout system, working with rows and columns simultaneously:

.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr); /* Three equal columns */
    grid-template-rows: 100px 200px;       /* Two rows of different heights */
    gap: 20px;                             /* Gap between grid items */
}

.item1 {
    grid-column: 1 / 3;                    /* Spans columns 1 to 2 */
    grid-row: 1 / 2;                       /* Only in the first row */
}

.item2 {
    grid-area: 2 / 1 / 3 / 4;              /* row-start/column-start/row-end/column-end */
}

Animations & Transitions

CSS transitions allow property changes to occur smoothly over a specified duration:

.button {
    background-color: blue;
    color: white;
    transition: background-color 0.3s ease, transform 0.3s ease;
}

.button:hover {
    background-color: darkblue;
    transform: scale(1.1);
}

CSS animations allow more complex animations with keyframes:

@keyframes pulse {
    0% {
        transform: scale(1);
    }
    50% {
        transform: scale(1.1);
    }
    100% {
        transform: scale(1);
    }
}

.pulsing-element {
    animation: pulse 2s infinite;
}

Custom Properties (Variables)

CSS variables (custom properties) allow you to store and reuse specific values:

:root {
    --primary-color: #3498db;
    --secondary-color: #2ecc71;
    --text-color: #333;
    --font-size-base: 16px;
}

.button {
    background-color: var(--primary-color);
    color: white;
    font-size: var(--font-size-base);
}

.alert {
    border: 2px solid var(--secondary-color);
    color: var(--text-color);
}

Dark Mode Implementation

Dark mode can be implemented using CSS variables and the prefers-color-scheme media query:

:root {
    --bg-color: #ffffff;
    --text-color: #333333;
}

@media (prefers-color-scheme: dark) {
    :root {
        --bg-color: #121212;
        --text-color: #f0f0f0;
    }
}

body {
    background-color: var(--bg-color);
    color: var(--text-color);
}

/* Or manually toggle with a class */
.dark-mode {
    --bg-color: #121212;
    --text-color: #f0f0f0;
}

Neon Effects & Styling

Creating neon effects with CSS:

.neon-text {
    color: #fff;
    text-shadow: 
        0 0 5px #fff,
        0 0 10px #fff,
        0 0 20px #ff00de,
        0 0 30px #ff00de,
        0 0 40px #ff00de;
}

.neon-border {
    border: 2px solid #08f;
    box-shadow: 
        0 0 5px #08f,
        0 0 10px #08f,
        inset 0 0 5px #08f,
        inset 0 0 10px #08f;
}

.neon-button {
    background-color: transparent;
    color: #0ff;
    border: 2px solid #0ff;
    text-shadow: 0 0 5px #0ff;
    box-shadow: 0 0 10px #0ff;
    transition: all 0.3s;
}

.neon-button:hover {
    background-color: #0ff;
    color: #000;
    box-shadow: 0 0 20px #0ff;
}
